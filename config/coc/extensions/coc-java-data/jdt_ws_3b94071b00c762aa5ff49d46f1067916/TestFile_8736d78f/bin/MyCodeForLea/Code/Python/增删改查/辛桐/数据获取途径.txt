可以在国家统计局、国防部、外交部等政府机构的官方网站上找到中国在经济、军事等方面的发展数据。此外，还可以在各大媒体的经济、军事板块中找到相关的报道和数据分析。

以下是一些常用的数据来源：

- 国家统计局：http://www.stats.gov.cn/
- 国防部：http://www.mod.gov.cn/
- 外交部：https://www.fmprc.gov.cn/
- 中国人民银行：http://www.pbc.gov.cn/
- 国家发展和改革委员会：http://www.ndrc.gov.cn/
- 中国证券监督管理委员会：http://www.csrc.gov.cn/
- 中国银行保险监督管理委员会：http://www.cbirc.gov.cn/
- 中国军事科学院：http://www.mss.gov.cn/

通过这些机构和网站，可以获取到中国在经济、军事等方面的数据和报告，例如国内生产总值、贸易额、武器装备数量、军事支出等数据。同时，一些研究机构和行业协会也会发布相关的数据和报告，可以通过搜索引擎或相关网站进行查询。
